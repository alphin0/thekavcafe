# KAV – A Green Bistro Website

## Overview
This is a premium, aesthetic website designed for **KAV – A Green Bistro**. It features a modern, nature-inspired design with a focus on serenity, eco-luxury, and freshness.

## Files
- **index.html**: The main structure of the website.
- **style.css**: Custom CSS using a premium color palette (Forest Green, Sage, Gold) and responsive design.
- **script.js**: Handles smooth scrolling, mobile menu, and scroll animations.
- **assets/**: Contains all images including generated hero/food images and existing bistro assets.

## How to Run
Simply open `index.html` in any modern web browser.

## Features
- **Hero Section**: Cinematic background with tagline.
- **About & Philosophy**: Storytelling sections with elegant typography.
- **Menu Highlights**: Card layout for signature dishes.
- **Gallery**: Masonry-style grid showcasing the bistro's ambience.
- **Contact & Location**: Interactive buttons for WhatsApp/Call and a reservation form.
- **Animations**: Subtle scroll-reveal effects.
